O que é HTML?

HTML é uma das linguagens de marcação mais conhecida e utiliza um formato simples de código, que pode ser gerado sem ajuda de aplicativos especiais

Para criar um documento HTML, tudo o que você precisa é de um editor de texto ASCII como o NotePad (Bloco de Notas), por exemplo. Se você precisar verificar a formatação do documento durante a criação, pode usar qualquer navegador, poois todos eles são capazer de interpretar o código.

Principais comando
